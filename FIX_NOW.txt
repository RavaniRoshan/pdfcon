================================================================================
                    ALTERNATIVE: QUICK MANUAL FIX
================================================================================

If the helper doesn't work, create .env file manually:

1. Create a new text file in this folder
2. Name it exactly: .env (with the dot, no .txt!)
3. Open it and type this line:
   DOPPIO_API_KEY=dp_your_actual_key_here

   (Replace dp_your_actual_key_here with your real key from doppio.sh)

4. Save and close the file
5. Run: python src\generator.py
